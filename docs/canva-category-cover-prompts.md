# Canva 赛事大类封面提示词

用途：为首页七个赛事大类生成 16:9 横版封面图。风格不要像课程或学习平台，要像体育赛事入口、赛事图鉴、体育内容平台。

通用要求：

- 真实场馆感，器材主体明显，灯光强烈。
- 主体图像优先，中文标题可以少量覆盖。
- 避免扁平图标、卡通、密集文字和课堂感。
- 每张图给标题留出安全区，方便网页卡片叠字。

## 七大类封面

- 球类赛事：足球入网特写，明亮体育场灯光，绿色草皮，旁边有篮球、排球、网球等器材氛围。
- 田径类：红色跑道、起跑器、跨栏、终点线，画面有速度拖影和冲刺感。
- 水上类：蓝色泳池水面、泳道线、跳台、水球或赛艇航道，突出水光反射。
- 冰雪类：雪道和冰面融合，滑雪板、冰球或冰壶石，冷色聚光灯和高速感。
- 格斗类：拳台、柔道垫或击剑道，拳套和护具在前景，聚光灯突出对抗感。
- 技巧评分类：体操赛台、花滑冰面、跳水或自由式滑雪动作轨迹，优雅、舞台感明显。
- 竞速类：赛车场弯道、自行车轮、短道速滑或赛艇冲刺线，高速运动模糊。
