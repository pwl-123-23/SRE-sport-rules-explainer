# Canva 赛事图卡设计提示词

用途：为体育赛事规则 APP 生成 16:9 横版赛事图卡。每张图卡建议包含：中文赛事名、所属大类标签、核心场地或器材、1-2 个规则关键词。统一风格为清晰教育型、扁平矢量、明亮但不刺眼、适合网页和小程序卡片。

通用提示词模板：

```text
为体育赛事规则APP设计一张【赛事名】赛事图卡，16:9横版，中文标题“【赛事名】”，小标签“【赛事大类】”。画面突出【场地/器材/动作】，加入规则关键词：【关键词】。风格：清晰教育型、扁平矢量、色彩明快、适合小程序和网页卡片，避免复杂背景和密集文字。
```

## 球类赛事

- 足球：草地球场、球门、罚球区；关键词：进球、越位、犯规。
- 篮球：木地板球场、篮筐、三分线；关键词：2分、3分、24秒。
- 排球：隔网球场、发球区、拦网；关键词：每球得分、轮转、触网。
- 网球：网球场、球拍、发球区；关键词：局、盘、抢七。
- 羽毛球：羽毛球场、球拍、单双打边线；关键词：21分、发球区、触网。
- 乒乓球：球台、球网、球拍；关键词：11分、发球、旋转。
- 手球：室内球场、球门区、7米线；关键词：射门、走步、球门区。
- 棒球：内场菱形、本垒、球棒；关键词：出局、跑垒、得分。

## 田径类

- 短跑：分道跑道、起跑器、终点线；关键词：起跑、分道、计时。
- 中长跑：标准跑道、多圈路线、集团跑；关键词：配速、并道、冲刺。
- 跨栏：跑道、栏架、起跨动作；关键词：栏间节奏、抢跑、串道。
- 接力：跑道、接力棒、交接区；关键词：交接棒、接力区、犯规。
- 跳远：助跑道、起跳板、沙坑；关键词：起跳线、丈量、犯规。
- 跳高：横杆、立柱、落地区；关键词：试跳、横杆、失败。
- 铅球：投掷圈、抵趾板、扇形落区；关键词：投掷、落点、出圈。
- 标枪：助跑道、标枪、扇形落区；关键词：出手、枪尖、落区。
- 马拉松：城市路线、补给站、终点拱门；关键词：42.195公里、配速、计时。

## 水上类

- 游泳：泳池泳道、出发台、水线；关键词：泳姿、触壁、犯规。
- 跳水：跳台、跳板、水池；关键词：难度、完成度、入水。
- 水球：泳池、球门、水球；关键词：进攻时间、射门、犯规。
- 花样游泳：泳池、同步动作、队形；关键词：同步、艺术表现、难度。
- 赛艇：水道、赛艇、桨；关键词：航道、冲线、计时。
- 皮划艇：静水赛道、皮艇或划艇、桨；关键词：航道、计时、器材。

## 冰雪类

- 滑雪：雪道、旗门、滑雪板；关键词：路线、速度、犯规。
- 滑冰：冰面、冰刀、弯道；关键词：滑行、道次、计时。
- 冰球：冰场、球门、球杆；关键词：进球、越位、身体冲撞。
- 雪车：冰道、雪车、弯道；关键词：出发、计时、线路。
- 雪橇：冰道、雪橇、低姿态滑行；关键词：出发、线路、计时。
- 冰壶：冰壶赛道、圆垒、冰壶石；关键词：投壶、刷冰、得分区。

## 格斗类

- 拳击：拳台、拳套、回合铃；关键词：回合、击中、读秒。
- 柔道：榻榻米、道服、投技；关键词：一本、技有、犯规。
- 跆拳道：护具、电子得分、踢击；关键词：有效击打、警告、回合。
- 摔跤：摔跤垫、红蓝选手、控制动作；关键词：压制、得分、出界。
- 击剑：剑道、面罩、剑；关键词：有效部位、优先权、得分灯。

## 技巧评分类

- 体操：器械、落地区、动作姿态；关键词：难度分、完成分、落地。
- 艺术体操：地毯、彩带或圈、身体难度；关键词：器械、艺术表现、完成度。
- 跳水：跳台、水池、空中翻腾；关键词：难度、入水、裁判评分。
- 自由式滑雪：雪坡、跳台、空中动作；关键词：难度、完成度、落地。
- 花滑：冰面、旋转、托举或跳跃；关键词：技术分、节目内容、摔倒扣分。

## 竞速类

- 自行车：赛道或公路、自行车、冲刺线；关键词：计时、集团、冲刺。
- 赛车：赛道、赛车、发车灯；关键词：圈速、维修区、旗语。
- 短道速滑：椭圆冰道、弯道超越、防护垫；关键词：超越、犯规、计时。
- 速度滑冰：长道冰场、分道、计时牌；关键词：道次、换道、计时。
- 赛艇：直线水道、赛艇、桨频；关键词：航道、冲线、计时。
